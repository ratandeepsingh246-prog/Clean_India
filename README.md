**Project Overview**
This project analyzes Swachh Bharat Mission solid waste management data published by the Government of India.
The objective is to transform raw public data into actionable insights using Power BI for state-level monitoring and district-level performance evaluation.

The project focuses on understanding legacy waste remediation, waste generation, processing capacity, and infrastructure gaps.
